# Java Card Game (OOP + Swing)
A complete, turn-based desktop card game developed in Java. This project serves as a practical application of Object-Oriented Programming (OOP) concepts, featuring an interactive event-driven GUI built with Java Swing.

Features
Turn-Based Gameplay: Compete against a computer opponent with automated decision-making logic.

Interactive GUI: Distinct screens for the Main Menu, active Gameplay, and Game Over states.

Core OOP Design: Architecture relies heavily on Encapsulation, Inheritance, Polymorphism, and Abstraction (utilizing modular Card, Deck, Player, and GameController classes).

Data Persistence: Local saving and loading of high scores, tracking player names, scores, and dates.

Robust State Management: Handles programmatic deck shuffling, turn validation, keyboard/mouse event listeners, and controlled game loop delays.

Language: Java
UI Library: Java Swing / AWT
Storage: Local File I/O

Getting Started:
1. Clone the repository.
2. Compile the Java source files or import the project into your preferred IDE (Eclipse, IntelliJ IDEA, etc.).
3. Run the main class to launch the Swing interface and start the game.
